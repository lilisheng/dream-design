### 与dom层级一致的组件配置

非嵌套组件只需遵守如下格式

```js
    {
    	tag: "el-button",
    	title: "el按钮",
            ...
        childrens: []
    }
```

即可渲染出如下dom

```vue
<el-button></el-button>
```

嵌套组件只需遵守如下格式

```js
    {
    	tag: "el-row",
    	title: "el行容器",
            ...
        childrens: [    
        	{
    			tag: "el-col",
    			title: "el列容器",
            	...
        		childrens: []
    		}]
    }
```

即可渲染出如下dom

```vue
<el-row>
    <el-col></el-col>
</el-row>
```
